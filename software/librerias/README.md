# Librerías

## Librerías Estándar (Arduino IDE)

Instalar desde el Library Manager de Arduino IDE:

1. **ESP8266WiFi** - v[1.0]
   - Propósito: Crear un servidor web para el proyecto
   - Instalación: `Sketch > Include Library > ESP8266WiFi`
2. **FirebaseClient** - v[2.2.9]
   - Propósito: Almacenar los datos en una base de datos en tiempo real
   - Instalación: `Sketch > Include Library > Manage Libraries > FirebaseClient`
